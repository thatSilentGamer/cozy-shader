# Cozy Shader

A warm, atmospheric Minecraft shader made with Iris.

## Features:
- Warm golden sunlight with sunrise and sunset color shifts
- Rich orange torchlight for a cozy indoor feel
- Animated water with depth-based coloring
- Directional shadows with distortion
- Distance fog
- Subtle vignette
- Gamma corrected lighting

## How to Install:
1. Install [Iris Shaders](https://irisshaders.dev)
2. Download Cozy Shader from the Modrinth page
3. Launch Minecraft with Iris
4. Go to Options > Video Settings > Shader Packs
5. Click Open Shader Pack Folder
6. Place the downloaded zip into that folder
7. Select Cozy Shader from the list and click Apply

## Requirements:
- Minecraft 1.21 or higher
- Iris Shaders
